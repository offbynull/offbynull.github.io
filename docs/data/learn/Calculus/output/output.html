Factoring Polynomials

To factor a quadratic ax2+bx+cax^2+bx+cax2+bx+c, find two numbers r and s such that when ...

  • added, ar+s=bar+s=bar+s=b.
  • multiplied, rs=crs=crs=c.

ax2+arx+sx+rs=ax(x+r)+s(x+r)=(ax+s)(x+r)ax^2+arx+sx+rs = ax(x+r)+s(x+r) = (ax+s)(x+r)ax2+arx+sx+rs=ax(x+r)+s(x+r)=(ax+s)(x+r)

For example, 2x2−7x−4=2x2−8x+1x−4=(2x+1)(x−4)2x^2-7x-4 = 2x^2-8x+1x-4 = (2x+1)(x-4)2x2−7x−4=2x2−8x+1x−4=(2x+1)(x−4).

Factoring quadratics is useful for finding their roots (where x=0) as well as algebraic manipulations such as simplifying.

You often see explained online for the special case where a=1: x2+bx+cx^2+bx+cx2+bx+c, which means you're trying to find two numbers r and s where r+s=br+s=br+s=b and rs=crs=crs=c...

  • added, equal b: r+s=br+s=br+s=b.
  • multiplied, equal c: rs=crs=crs=c.

x2+rx+sx+rs=x(x+r)+s(x+s)=(x+r)(x+s)x^2+rx+sx+rs = x(x+r)+s(x+s) = (x+r)(x+s)x2+rx+sx+rs=x(x+r)+s(x+s)=(x+r)(x+s).

Expressions that follow the pattern ...

  • a2−b2a^2-b^2a2−b2, called difference of squares, factor as a2−b2=(a−b)(a+b)a^2-b^2 = (a-b)(a+b)a2−b2=(a−b)(a+b).
  • a3−b3a^3-b^3a3−b3, called difference of cubes, factor as a3−b3=(a−b)(a2+ab+b2)a^3-b^3 = (a-b)(a^2+ab+b^2)a3−b3=(a−b)(a2+ab+b2).

Completing the Square

To complete the square for a ax2+bx+cax^2+bx+cax2+bx+c, begin by pulling out a from first two terms: a(x2+bax)+ca(x^2+\frac{b}{a}x)+ca(x2+ab​x)+c.

Then, rewrite the inner group x2+baxx^2+\frac{b}{a}xx2+ab​x to be in the form (x+s)2(x+s)^2(x+s)2. To do this, a constant term needs to be added to the inner group such that it can be factored. (x+s)2(x+s)^2(x+s)2 expands out to x2+2sx+s2x^2+2sx+s^2x2+2sx+s2, meaning the constant term should map to s2s^2s2.

To determine what s should be for the inner group, use algebra. It's known that the ...

  • inner group's middle term ab\frac{a}{b}ba​ maps to 2sx2sx2sx.
  • inner group's constant term should map to s2s^2s2.
bax=2sxba=2sb2a=s\frac{b}{a}x = 2sx \frac{b}{a} = 2s \frac{b}{2a} = s ab​x=2sxab​=2s2ab​=s

Since s = \frac{b}{2a}, adding b2a2\frac{b}{2a}^22ab​2 to the inner group makes it factorable into the form (x+s)2(x+s)^2(x+s)2: a(x2+bax+b2a2)+ca(x^2+\frac{b}{a}x+\frac{b}{2a}^2)+ca(x2+ab​x+2ab​2)+c. However, the problem with adding this term to the inner group is that the overall expression is now different from the original: a(x2+bax)+c≠a(x2+bax+b2a)+ca(x^2+\frac{b}{a}x)+c \neq a(x^2+\frac{b}{a}x+\frac{b}{2a})+ca(x2+ab​x)+c=a(x2+ab​x+2ab​)+c. A quantity of a(b2a2)a(\frac{b}{2a}^2)a(2ab​2) has been added in, and so that same quantity needs to be removed: a(x2+bax)+c=a(x2+bax+b2a2)+c−a(b2a2)a(x^2+\frac{b}{a}x)+c = a(x^2+\frac{b}{a}x+\frac{b}{2a}^2)+c-a(\frac{b}{2a}^2)a(x2+ab​x)+c=a(x2+ab​x+2ab​2)+c−a(2ab​2).

For example, to complete the square for 2x2−12x+112x^2-12x+112x2−12x+11 ...

  • pull out a from first two terms: 2(x2−6x)+112(x^2-6x)+112(x2−6x)+11.

  • factor inner group to (x+s)2(x+s)^2(x+s)2 form: 2(x−3)2+11−182(x-3)^2+11-182(x−3)2+11−18.
  • add together constant terms: 2(x−3)2−72(x-3)^2-72(x−3)2−7.

Completing the square is useful because it gets your equation into graph transformation format. For the example above, just by looking at the completed formula 2(x−3)2−72(x-3)^2-72(x−3)2−7 you can tell that it's a parabola ...

  • shifted down by 7.
  • shifted right by 3.
  • stretched up by 2 times.

Polynomial Division

Polynomial division is a process that breaks a polynomial into two factors, given that one of the factors is already known. The process works similarly to long division in that you're dividing the polynomial by that known factor to determine the other factor. For example, consider the polynomial 2x3+x2−6x−82x^3+x^2-6x-82x3+x2−6x−8. If you already know that (x−2)(x-2)(x−2) is a factor, you can divide the polynomial by that known factor to determine the other factor:

               2x^2  + 5x  + 4
    .-------------------------
x-2 |    2x^3 +  x^2 -  6x - 8
      - (2x^3 - 4x^2)
	    -------------
		        5x^2 -  6x
			 - (5x^2 - 10x)
			   ------------
			            4x - 8
				     - (4x - 8)
					  ---------
					         0

⚠️NOTE️️️⚠️

Note what's happening here. The first term in the quotient is 2x22x^22x2 because when you multiply it by the divisor (x−2)(x-2)(x−2) it results in 2x3+x22x^3+x^22x3+x2. Subtracting 2x3+x22x^3+x^22x3+x2 from the dividend completely eliminates the 2x^3 from the dividend.

This same process continues for each term in the quotient.

In the example, polynomial division determined that the other factor is 2x2+5x+42x^2+5x+42x2+5x+4. That is, (x−2)(2x2+5x+4)=2x3+x2−6x−8(x-2)(2x^2+5x+4)=2x^3+x^2-6x-8(x−2)(2x2+5x+4)=2x3+x2−6x−8. The other factor can then go on to be factored again by some other method (or potentially the same method), assuming it's factorable.

For polynomial division to work, you must already know one of the polynomial's factors. If you don't already have a starting factor, there are multiple ways to find one:

  • Graphing calculator: Try using a graphing calculator to see if you can find one of the roots. The roots are values of x where the graph touches or crosses the x-axis.

  • Trial-and-error: Try plugging values for x into the polynomial until one evaluates to 0. For example, 2x3+x2−6x−82x^3+x^2-6x-82x3+x2−6x−8 when ...

    • x=0 evaluates to -8
    • x=1 evaluates to -11
    • x=2 evaluates to 0

    When x=2, the polynomial evaluates to 0, meaning (x−2)(x-2)(x−2) is a factor.

  • Rational root theorem: Assuming P(x) is a polynomial with integer coefficients, a root of the equation P(x)=0P(x)=0P(x)=0 can be expressed in the form pq\frac{p}{q}qp​, where ...

    • p is a factor of the constant term.
    • q is a factor of the leading coefficient.

    In other words, you can find a factor by testing all possible pq\frac{p}{q}qp​ to see if it's a root / factor of the polynomial. For example, the polynomial P(x)=2x3+x2−6x−8P(x)=2x^3+x^2-6x-8P(x)=2x3+x2−6x−8 has ...

    • constant term factors p={-8,-6,-4,-2,-1,1,2,4,6,8}.
    • leading coefficient factors q={1,2}.

    Testing possible pq\frac{p}{q}qp​ within P(x) shows that 42=2\frac{4}{2}=224​=2 is a root: P(2)=0P(2) = 0P(2)=0. Since 222 is a root, (x−2)(x-2)(x−2) is a factor.

    ⚠️NOTE️️️⚠️

    It's called "rational root theorem" because the coefficients have to be rational / real numbered coefficients? (e.g. coefficients can't be an imaginary or complex number).

You can use the methods described above to iteratively pull out roots using polynomial division until the entire polynomial is factored. For example, ...

  1. use rational root theorem to determine (x-2) is a factor of x3−3x2−10x+24x^3-3x^2-10x+24x3−3x2−10x+24.
  2. use polynomial division to factorize x3−3x2−10x+24x^3-3x^2-10x+24x3−3x2−10x+24 to (x−2)(x2−x−12)(x-2)(x^2-x-12)(x−2)(x2−x−12).
  3. use rational root theorem to determine (x-4) is a factor of x2−x−12x^2-x-12x2−x−12.
  4. use polynomial division to factorize x2−x−12x^2-x-12x2−x−12 to (x−2)(x−4)(x-2)(x-4)(x−2)(x−4).

x3−3x2−10x+24=(x−2)(x−2)(x−4)x^3-3x^2-10x+24 = (x-2)(x-2)(x-4)x3−3x2−10x+24=(x−2)(x−2)(x−4).

Terminology

  • quadratic - An expression where the highest exponent on the top-level terms is 2. For example, 2x2+4x+42x^2+4x+42x2+4x+4 is a quadratic.

  • root - Given a polynomial P(x), its roots are x values where P(x)=0P(x)=0P(x)=0. If you were to graph P(x), the roots would be values of x that touch the x-axis.

    • b2−4ac>0b^2-4ac > 0b2−4ac>0, there are two real roots.
